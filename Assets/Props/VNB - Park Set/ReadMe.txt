The aim for this package is to just set-up things quickly, by providing ready-to-use prefabs of everything. This includes:

- Materials and original palette. (if you're using URP, then you will need to convert these to URP, this is done with literally just one click from within Unity)
- Prefabs of models ready to place in scene, including certain ones such as the basket, swings with different variants and a bike with separated parts such as the wheels, handling and pedals, all ready to be rotated or manipulated from script or just the inspector itself.

*Make sure to snap things to the grid when using tiles*

*Look at my itch.io page for details*

[[ ENG ]]

--- Description ---

This set is based on some kind of natural park, you have benches, picnic stuff, fences, and so on. The idea behind everything comes from a voxel model pack featuring the same theme, made by a friend (link in the main post on itch.io).

It's a pretty simple set, tried to optimize it as much as possible, so one can use it for mobile, PC or whatever (it's low poly after all), and comes with the palette I used originally, which is a single 12x12 pixels image, you can just change the color pixels to whatever you want in order to create new ones, some parts are designed to be easily adjusted to be used when creating games (the bike, for example, features certain parts separated to be animated or used individually).

----------
Credits to VNB (Leo) (https://vnbp.itch.io/)
----------

You're free to use every model the way you want for any commercial / non-commercial project. (comments and mentions are always welcome and certainly appreciated)

Please make sure to review the asset on itch.io if you found it useful, leave a comment there, along with any kind of suggestion (or problem you faced while using it) in order to improve it!

------

[Notes]
* If you want to use it on Unity, for example, you may find it useful to tweak certain options when importing the image/texture in order to properly import it correctly.
* The light models do feature a SECOND material, which refers to the light itself. This way you can set it to any color you want (and maybe use a different emmisive material only for the light).
(Same with the flower ones containing MAT in the name, with 3 materials used in order to change the color, you can still use the simple ones and different palettes)





[[ ESP ]]

--- Descripción ---

El set esta ambientado bajo la tematica "parque", hay modelos de bancos, comida, castillos de arena para niños, etc.. La idea original proviene de un set realizado por una amiga (en estilo voxel, encontraran el link en mi post original en itch.io).

Es un set bastante simple, low poly, que trate de modelar de forma optimizada. Utiliza una paleta de 12x12 pixeles (colores), que puedes modificar para crear nuevas variantes. Algunos modelos fueron diseñados de forma que puedan manipularse de forma sencilla y animarse por separado (la bici se divide en partes, por lo las ruedas pueden ser animadas por separado, al igual que las canastas y el volante).

----------
Creditos a VNB (Leo) (https://vnbp.itch.io/)
----------

Eres libre de usar los modelos de la forma que quieras ya sea comercial o no comercialmente. (las menciones son siempre bienvenidas)

Sientete libre de dejarme una review en itch.io si el pack te ha sido útil, deja un comentario junto con cualquier tipo de sugerencia (o problema) a modo de poder mejorarlo!

----

[Notas]
* Si utilizas los modelos en Unity, por ejemplo, existe la probabilidad de que debas jugar con las opciones a la hora de importar la imagen/textura de forma correcta.
* Los modelos de luces contienen un SEGUNDO material, el cual referencia solo a la luz, por lo que de esta forma puedes cambiar el color de la misma, y utilizar un material distinto (por ejemplo, con emisión).
(Mismo caso con las flores que tienen MAT en el nombre, con 3 materiales para ajustar colores por este medio)
